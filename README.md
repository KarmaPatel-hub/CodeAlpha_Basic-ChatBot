# Basic ChatBot 
A simple rule-based chatbot made using Python.

##  Project Description

This chatbot can respond to basic user messages using predefined replies.  
It is built using Python and uses simple programming concepts like:
- Classes & Objects
- Functions
- if-elif conditions
- Loops
- User Input/Output

##  Features

- Responds to greetings
- Answers simple questions
- Friendly chatbot replies
- Exit chat using "bye"

##  Technologies Used

- Python
